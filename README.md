# Student Expense Tracker Dashboard

This repository contains a project to track and analyze monthly student expenses and savings. The goal is to help students gain insights into their spending patterns and identify opportunities for budgeting and saving.

## Project Overview

The student expense tracker dashboard processes monthly spending data (rent, groceries, transport, entertainment, subscriptions, etc.) and presents interactive visualizations and metrics. It also includes simple projections to estimate savings based on different spending scenarios.

## Features

- Data cleaning and preprocessing of expense data.
- Exploratory data analysis (EDA) to understand spending distributions.
- Dashboard built with Streamlit to visualize expenses by category, month, and total spending.
- Savings projection and overspending warnings.
- Modular Python codebase for easy extension.

## Repository Structure

- **data/** – stores raw and processed data files.
- **notebooks/** – Jupyter notebooks for EDA and prototyping analyses.
- **src/** – source code for data processing, analysis, and Streamlit app.
- **images/** – images and plots used in documentation or the app.

## Getting Started

To run the dashboard locally:

1. Clone this repository.
2. Install dependencies from `requirements.txt`.
33. (Optional) Use the provided sample dataset `data/sample_expenses.csv` to explore the dashboard.
4. Run the Streamlit app:

       streamlit run streamlit_app.py

5. Access the dashboard in your browser.
